# Add policy safely

Допоможи додати новий Windows registry/policy tweak у Win11 25H2 Calm Mode.

Перед кодом:
1. Поясни, що саме робить policy.
2. Вкажи registry path, name, type, desired value.
3. Скажи, чи це policy-backed, UISetting, BestEffort, DeprecatedOrLegacy або MaybeIgnoredOnEdition.
4. Опиши ризики.
5. Скажи, чи треба додати path у registry backup.
6. Скажи, що треба оновити в README і changelog.

Потім запропонуй мінімальний patch без повного rewrite.

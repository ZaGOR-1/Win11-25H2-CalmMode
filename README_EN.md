# Win11 25H2 Calm Mode

<p align="right">
  <a href="README.md"><strong>Українська</strong></a>
</p>

A PowerShell script for carefully tuning Windows 11 25H2 into a calmer mode: fewer Copilot/AI features, Widgets, ads, recommendations, Edge background processes, automatic Windows Update drivers, and noisy UI prompts.

This is **not an aggressive debloater**. It does not disable Defender, Firewall, the Windows Update service, Microsoft Store, WebView2, .NET, certificates, or critical system services.

> The idea is simple: do not break Windows; set clear policy/registry values, create backups, show reports, and keep rollback understandable.

---

## Quick Start

```powershell
Set-ExecutionPolicy -Scope Process Bypass -Force
cd "$env:USERPROFILE\Desktop"

# 1. Safe audit: changes nothing
.\Win11-25H2-CalmMode.ps1 -Mode Audit

# 2. Apply: must be run as Administrator
.\Win11-25H2-CalmMode.ps1 -Mode Apply

# 3. Verify after reboot
.\Win11-25H2-CalmMode.ps1 -Mode Verify
```

The default mode is safe: running without parameters is the same as `-Mode Audit`.

---

## GUI

The easiest way to launch the GUI is to double-click:

```text
Win11-25H2-CalmMode-GUI.cmd
```

The GUI includes a two-pane **categories -> tweaks** list, badges for `Official` / `UISetting` / `BestEffort` / `Deprecated`, a selected-tweak detail panel, an **EN / UA** language toggle, safe **Audit**, **Summary / Details / Raw log** result tabs, confirmed **Apply** through UAC after a successful Audit, JSON config save/load, and **Undo last Apply** through `rollback.reg`.

---

## Safety Model

- `Audit` and `Verify` are read-only.
- `Apply` requires Administrator.
- Registry backup and `rollback.reg` are created before changes.
- A restore point is created when System Protection is enabled.
- Appx cleanup is off by default and opt-in only.
- Reports are written as `HTML`, `CSV`, and `JSON`.
- Build/edition limitations are shown honestly in the report.

Get-FileHash .\Win11-25H2-CalmMode-v2.11.zip -Algorithm SHA256

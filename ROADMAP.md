## Фаза 1 — Довіра до Apply-шляху (найбільша реальна прогалина) ✅🟡

**Мета:** уперше **живий**, документований прогін `Apply`+відкату в чистій VM.

**Обсяг (ручний прогін у VM, зняти звіти/скріншоти):**
- [ ] `Audit` без адміна → потім з адміном (очікувано: Warning про не-Win11 лише поза 25H2; зміни — 0).
- [ ] `Apply` → перевірити backup, restore point, `rollback.reg`, `gpupdate`, перезапуск Explorer.
- [ ] **Повторний `Apply` поспіль** → усе має бути `AlreadyConfigured`/`Compliant`, **не** `Changed`
  (жива перевірка ідемпотентності).
- [ ] reboot → `Verify` (застосоване = `VerifyOK`; чесно пояснити будь-який `VerifyFail` по edition/build).
- [ ] `-RestoreFrom <папка>` (`reg import`) → reboot → `Verify` (значення відкотились).
- [ ] GUI smoke: «Run Audit», «Undo last Apply».
- [ ] Результати → `docs/vm-validation.md`.

**Критерій готовності:** `docs/vm-validation.md` із заповненою таблицею (крок / очікувано / факт / pass-fail).

**Чи треба:** ✅ так — це єдине, що **матеріально** підвищує довіру до інструмента, який змінює систему.
Unit-тести вже покривають логіку (зокрема ідемпотентність), але живий запис+відкат жодного разу не
підтверджений. Обмеження: потребує VM і ручної роботи; у CI не автоматизується (reboot розриває скрипт). 🟡

## Фаза 2 — Нові «calm» політики → `v2.11` ✅🟢 (рекомендований набір зроблено, реліз — у Фазі R)

**Мета:** додати ще документованих політик проти нав'язування, **без** ламання Windows.

> **Дисципліна `add-policy`:** кожен ключ звірено з Microsoft Learn / ADMX перед додаванням
> (точний Path/Name/Type/Value, `MinBuild`/`Editions`, чесний `Confidence`, тест, README + CHANGELOG).
> Додано **7** настройок; каталог: 112 → **119** твіків. Гейти: Pester **76/76**, PSSA CLEAN, parse OK.

**Рекомендований перший набір (3b + 3c + безпечні з 3a) — зроблено:**

**3a. Edge quiet mode — розширення** (блок `EdgeQuietMode`)
- [x] `ShowRecommendationsEnabled=0` — підказки/рекомендації фіч Edge. *`Official`, Edge 89+.*
- [x] `EdgeShoppingAssistantEnabled=0` — shopping/купони-попапи. *`Official`, Edge 87+.*
- [x] `PersonalizationReportingEnabled=0` — менше трекінгу для реклами. *`Official`, Edge 80+.*
- [ ] `SpotlightExperiencesAndRecommendationsEnabled=0` — Edge spotlight. *Середня — лишено на потім.*
- [ ] `NewTabPageContentEnabled=0` — прибрати MSN-стрічку на новій вкладці. *Змінює NTP → opt-in, на потім.*

**3b. Історія активності / Timeline** (блок `Privacy`, `HKLM\SOFTWARE\Policies\Microsoft\Windows\System`)
- [x] `EnableActivityFeed=0`, `PublishUserActivities=0`, `UploadUserActivities=0` — `OSPolicy.admx`, `Official`,
  Pro+. **Чесна нотатка в коді/докум.:** фіча Activity History прибрана на 24H2/25H2 → ефект обмежений,
  хоча значення політик виставляються надійно.

**3c. Онлайн-контент у Settings** (блок `Privacy`)
- [x] `AllowOnlineTips=0` — **виправлено шлях** на `HKLM\SOFTWARE\Microsoft\Windows\CurrentVersion\Policies\Explorer`
  (не `Policies\Microsoft\Windows\Explorer`). `ControlPanel.admx`, але не CSP-нода → чесно `BestEffort`.

**3d. Пошук на таскбарі — policy-версія** (🟡, opt-in) — **НЕ зроблено (лишено на потім):**
- [ ] `ConfigureSearchOnTaskbarMode` (24H2+, Pro+) — policy-аналог наявного UI-твіка `SearchboxTaskbarMode`;
  жорсткіша (блокує перемикач у Settings) → **тільки opt-in**, не за замовчуванням.

**Лишилось (опційно, окремий пакет):** 2 середні з 3a (`SpotlightExperiencesAndRecommendationsEnabled`,
`NewTabPageContentEnabled`) + 3d. Робити, якщо захочеться; кожен — знову через `add-policy`.

**Реліз:** `v2.11` поки **локальний** (не закомічено). Тег/ZIP — разом із `v2.10` у Фазі R.

> **Свідомо НЕ кандидати** (це вже ламання функціоналу): `DisableSearch=1` (повне вимкнення пошуку),
> масове прибирання індексації, вимкнення clipboard history.

## ❌ Свідомо НЕ робимо

- **Read-only перевірка нової версії через мережу.** Мережева залежність заради косметики — суперечить
  «без зовнішніх залежностей»; релізи й так видно на GitHub.
- **Рефакторинг рушія на модулі.** Моноліт робочий, покритий **76** тестами, роздається одним `.ps1`.
  Розбивка = ризик без користувацької цінності (CLAUDE.md: «не роби rewrite без потреби»).
- **Компільований `.exe` / base64 / завантаження коду** — проти філософії (AV/SmartScreen, прозорість).
- Будь-що із забороненого списку нижче.
